# 🏅 Projeto Dados de Atletas

## 📌 Descrição

Este projeto foi desenvolvido em JavaScript e tem como objetivo representar um atleta através de uma classe, calculando informações importantes como categoria, IMC e média válida das notas.

---

## 🚀 Tecnologias utilizadas

* JavaScript
* Node.js

---

## 📂 Estrutura do projeto

* `dados-atletas.js` → Código principal do projeto
* `README.md` → Documentação

---

## ⚙️ Funcionalidades

* Calcular categoria do atleta com base na idade
* Calcular IMC (Índice de Massa Corporal)
* Calcular média válida (descartando maior e menor nota)
* Exibir todas as informações do atleta

---
